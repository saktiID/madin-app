<h1 align="center">MadinApp</h1>

## Tentang MadinApp

MadinApp merupakan aplikasi management untuk madrasah diniyyah (Madin) dengan fitur mengelolah beberapa data dalam administrasi tata usaha madrasah seperti:

-   Identitas Madin
-   Data Asatidz
-   Data Santri
-   Data Pelajaran
-   Data Kelas
-   Data Periode
-   Penilaian
-   Raport
